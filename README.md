SIT305-Task9.1p Lost and Found Maps App

This app allows users to manage lost and found items with Google Maps integration. Users can create, view, filter, and remove items, select locations using Google Places Autocomplete, and display nearby items on a map. The app uses SQLite database for local data storage and RecyclerView for displaying data.

Users can create a new lost or found advert with category, name, phone, description, date, and location

Each post requires an image upload

Users can select locations using Google Places Autocomplete

Users can get their current location using GPS location services

View all items in a list using RecyclerView

Filter items by category (Electronics, Pets, Wallets)

View item details on a separate screen

Remove items from the list

Each post includes a timestamp to show when it was created

Display lost and found items on Google Maps using markers

Radius-based filtering is used to display only nearby items on the map

Technologies Used: Android Studio, Java, XML, SQLite Database, RecyclerView, Google Maps SDK, Google Places API, Activity Result API, Fused Location Provider
